Reveal

The file bar.txt actually has a newline as the first character of its filename. 

            $ ls -l files/lighttpd/
            total 8
            -rw-r--r-- 1 colin colin 4 Feb 18 17:29 ''$'\n''bar.txt'
            -rw-r--r-- 1 colin colin 4 Feb 18 17:29 foo.txt

The escaping used by ls is hard to read, so here's the same thing using printf 

            $ for name in files/lighttpd/*.txt; do printf '%q\n' "$name"; done
            $'files/lighttpd/\nbar.txt'
            files/lighttpd/foo.txt
